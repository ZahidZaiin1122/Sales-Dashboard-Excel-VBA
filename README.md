# 📊 Interactive Sales Dashboard — Excel (Macros & Slicers)

## Project Overview
An interactive Excel dashboard built to analyze business and sales performance KPIs,
featuring automated reporting via VBA macros and dynamic filtering through slicers.

## 🗓️ Project Date
January 2026

## 🎯 Key Features
- **KPI Dashboard** — Visual summary of core sales and business performance metrics
- **VBA Automation** — Macros to automate repetitive tasks and one-click report refresh
- **Dynamic Filtering** — Pivot Tables, Pivot Charts, and slicers for drill-down analysis
- **Data Preparation** — Raw data cleaned and transformed using Power Query and advanced formulas
- **Professional Design** — Conditional formatting and dashboard best practices for clear insights

## 🛠️ Tools & Technologies
- Microsoft Excel (Advanced)
- Excel VBA / Macros
- Pivot Tables & Pivot Charts
- Power Query
- Slicers & Timeline Filters
- Conditional Formatting
